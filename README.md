# Stickman Ragdoll

A physics-based stickman ragdoll simulation that you can throw around in a box!

## Live Demo

🔗 **https://arnsgaming8.github.io/stickman-ragdoll/**

## How to Play

1. **Click and hold** on any body part (head, arms, legs, torso)
2. **Drag** to move or aim your throw
3. **Release** to throw the ragdoll with momentum
4. Watch it bounce around the box!

## Features

- 🔥 Real-time physics simulation
- 🖱️ Click and drag any body part
- 💥 Throw with momentum and velocity
- 🎯 Bouncy collisions with box walls
- 🌙 Dark theme with neon accents

## Technology

- Pure HTML5 Canvas
- Vanilla JavaScript (no libraries)
- Verlet integration for physics
- Distance constraints for ragdoll joints

## Development

The project is a single HTML file with embedded CSS and JavaScript:

```
index.html    - Main application (open in any browser)
```

To run locally:
1. Clone the repository
2. Open `index.html` in your browser

## Deployment

Automatic deployment to GitHub Pages via GitHub Actions on every push to master.
